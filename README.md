# 智能学习计划管理系统

## 项目概述

这是一个基于React + TypeScript + Vite构建的智能学习计划管理系统，包含学生端和管理员端两个主要模块。系统提供课程管理、学习计划生成、进度跟踪等功能。

## 技术栈

### 前端
- **框架**: React 18 + TypeScript
- **构建工具**: Vite
- **状态管理**: 自定义Store模式
- **路由**: Hash-based 路由
- **样式**: CSS-in-JS

### 后端
- **框架**: Django
- **数据库**: SQLite (开发环境)
- **API**: RESTful API

## 功能模块

### 学生端功能
1. **用户认证**
   - 学生注册/登录
   - 路由保护机制
   - Token持久化

2. **个人资料管理**
   - 头像上传和显示
   - 个人信息编辑
   - 偏好设置

3. **课程管理**
   - 查看可选课程
   - 加入/退出课程
   - 课程详情查看

4. **学习计划**
   - AI生成学习计划
   - 周计划视图
   - 进度跟踪

### 管理员端功能
1. **管理员认证**
   - 管理员注册/登录
   - 独立的路由保护

2. **仪表板**
   - 数据统计展示
   - 课程管理概览
   - 学生状态监控

3. **课程管理**
   - 创建/编辑课程
   - 学生进度查看
   - 任务管理

## 页面结构和样式说明

### 学生端页面

#### 学生主页 (StudentHome)
- **布局**: 侧边栏导航 + 主内容区
- **功能**: 课程概览、DDL提醒、快速导航
- **样式**: 蓝色主题，现代化卡片设计

#### 学生个人资料 (StudentProfile)
- **布局**: 表单式布局
- **功能**: 头像上传、信息编辑、偏好设置
- **样式**: 简洁表单设计，响应式布局

#### 学习计划 (StudentPlan)
- **布局**: 周视图网格
- **功能**: 每日任务展示、计划生成、进度跟踪
- **样式**: 日历式布局，彩色状态指示

### 管理员端页面

#### 管理员主页 (AdminHome)
- **布局**: 侧边栏 + 统计卡片 + 课程网格
- **功能**: 数据统计、课程管理、学生监控
- **样式**: 紫色主题，专业仪表板设计

## 数据交互

### 本地存储
- **用户信息**: `localStorage` 存储当前用户数据
- **认证状态**: Token和登录时间管理
- **课程数据**: 缓存课程信息提高加载速度

### API接口
- **认证接口**: `/api/auth/`
- **课程接口**: `/api/courses/`
- **计划接口**: `/api/plans/`
- **偏好接口**: `/api/preferences/`

## 路由保护机制

系统实现了完整的路由保护：
- 未登录用户访问受保护页面自动跳转登录页
- Token过期自动清理本地存储
- 异地登录检测和强制登出

## 开发指南

### 前端开发
```bash
cd front_end
npm install
npm run dev    # 开发服务器
npm run build  # 生产构建
```

### 后端开发
```bash
cd django_backend
pip install -r requirements.txt
python manage.py runserver
```
